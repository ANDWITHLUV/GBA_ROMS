# LEVEL UP
This ROM is a percentage calculator that calculates percentages in the range of [0,1.99] for integer and time format entries. Integer entries can range from [0,999] and time format ranges from [00:00,59:59]. The first screen is the title screen. It has a use counter at the bottom that ranges from [0,10,000]. Press start to move onto the main app screen. If you press select nothing will happen. The main application screen has three parts:
1. Option selection
2. Digit Selection
3. Save/Load Selection
